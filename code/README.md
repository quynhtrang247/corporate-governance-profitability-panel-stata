# Code

This folder contains the Stata commands used in the project **Relationship between Corporate Governance and Firm Profitability**.

## File
- `stata_commands.do`: command file used for data cleaning, variable construction, model diagnostics, panel regression, and export of results

## Main Tasks Covered
- Data cleaning and type conversion
- Variable construction
- Log transformation and winsorization
- Panel-data setup
- Model diagnostics
- Fixed Effects regression with clustered robust standard errors
- Export of regression tables
